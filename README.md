# critter-website

Single-page pre-launch website for Critter Engine.

## Local preview

Open `index.html` in a browser.

The updates signup endpoint requires an HTTP serverless runtime. Opening
`updates/index.html` directly as a `file://` URL can preview the page, but it cannot submit to
`/api/subscribe`. Test the signup through the deployed site, Cloudflare Pages, or Netlify dev.
